# XNAT-PIC

XNAT for Preclinical Imaging Centers (XNAT-PIC) is a free and open-source Windows desktop application, which offers several tools to expand the XNAT core functionalities to support
the preclinical imaging community and to promote open science practices.

# About

<!-- ![XNAT-PIC](images/XNAT-PIC.png) -->
<img src="assets/images/Workflow.PNG" width="750" title="XNAT-PIC">

**XNAT for Preclinical Imaging Centers (XNAT-PIC)** has been developed to expand XNAT's basic functionalities to preclinical imaging and consists of:

1. **XNAT-PIC Converter** to convert MRI Bruker and Optical Imaging IVIS raw data to DICOM standard, including DICOM tags for new MRI modalities, such as Chemical Exchange Saturation Transfer (CEST)
2. **Uploader** to easily import DICOM image datasets into the XNAT platform. It supports upload of whole projects, subjects and experiments (scans)
3. **Grouping Annotation Interface** to efficiently cope with different experimental protocols by labelling subjects with dedicated Custom Variables to manage several types of cohorts (e.g. treated/untreated, timepoints, doses, etc..)
